Resply Widget Demo

Widget de chat embeddable para tu sitio web. Instalación en menos de 2 minutos.

👋 ¡El widget está activo en esta página!
Haz clic en el botón de chat en la esquina inferior derecha para probarlo.

📦 Instalación Rápida

Copia y pega este código antes del cierre de la etiqueta </body> en tu HTML:

<script src="https://resply.vercel.app/widget/resply-widget.js"></script> <script> ResplyWidget.init({ workspaceId: 'TU_WORKSPACE_ID', primaryColor: '#667eea', position: 'bottom-right' }); </script>

⚙️ Configuración

Opciones disponibles para personalizar el widget:

  • workspaceId (requerido): Tu ID de workspace desde el dashboard
  • primaryColor (opcional): Color principal en formato hexadecimal (default: #667eea)
  • position (opcional): 'bottom-right' o 'bottom-left' (default: bottom-right)
  • apiUrl (opcional): URL de la API (default: https://resply.vercel.app)

🌐 Instalación por Plataforma

WordPress

Opción 1: Editor de temas

  1. Ve a Apariencia → Editor de temas
  2. Edita el archivo footer.php
  3. Pega el código antes de </body>

Opción 2: Plugin

  1. Instala el plugin "Insert Headers and Footers"
  2. Ve a Settings → Insert Headers and Footers
  3. Pega el código en "Scripts in Footer"

Shopify

  1. Ve a Online Store → Themes
  2. Haz clic en Actions → Edit code
  3. Abre el archivo theme.liquid
  4. Pega el código antes de </body>
  5. Guarda los cambios

Webflow

  1. Ve a Project Settings
  2. Haz clic en la pestaña Custom Code
  3. Pega el código en "Footer Code"
  4. Publica tu sitio

React / Next.js

Opción 1: Componente con useEffect

useEffect(() => { const script = document.createElement('script'); script.src = 'https://resply.vercel.app/widget/resply-widget.js'; script.async = true; document.body.appendChild(script); script.onload = () => { (window as any).ResplyWidget.init({ workspaceId: 'TU_WORKSPACE_ID', primaryColor: '#667eea', position: 'bottom-right' }); }; return () => { document.body.removeChild(script); }; }, []);
💡 Nota: Reemplaza TU_WORKSPACE_ID con tu workspace ID real desde el dashboard de Resply.
